The following set of functions are formatters to be used according to the Format module of the OCaml standard lib. They are used to print the various kind of nodes that constitute a term graph, as well as pointers. They take in input a pool in order to mark in the output the unevaluated subterms using angle brackets and a subscript which is the position of the job in the pool (e.g. !<m‚ÇÅ>‚ÇÇ is a value whose approximant is !<>‚ÇÇ and such that the job number 2 required to reduce m‚ÇÅ; equivalently it is a term graph whose pool contains in second position a pointer to the occurrence of m‚ÇÅ inside the value)
