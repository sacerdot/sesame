Module Sesame.REPL

It implements a REPL loop: the user is asked to enter a term that is parsed, checked to be proper and evaluated, until the user quits the REPL loop.

val test : string -> unit

Given a string s, it parses the string, checks that the parsed term is proper and then evaluates it, throwing away the computed normal form.
